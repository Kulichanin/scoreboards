# Scoreboards

## Обзор

Хочется усложнить данное задание работой с БД Postgress. Проект назовем Scoreboards.
Вывод данные на страничку о перелетах на сейчас согласно настройкам.
Сверху окно авторизаци для тех кому нужно посмотреть.
Самое сложное панель настроек информации:

- откуда
- куда
- за какой период

### БД

Для получения БД следующая команда

Данные по полётам за три месяца (размер БД примерно 1,3 Гбайт)

```bash
wget -P test/data https://edu.postgrespro.ru/demo-20250901-3m.sql.gz
```

Данные по полётам за шесть месяцев (размер БД примерно 2,7 Гбайт)

```bash
wget -P test/data https://edu.postgrespro.ru/demo-20250901-6m.sql.gz
```

Данные по полётам за год (размер БД примерно 5,4 Гбайт)

```bash
wget -P test/data https://edu.postgrespro.ru/demo-20250901-1y.sql.gz
```

Данные по полётам за год (размер БД примерно 5,4 Гбайт)

```bash
wget -P test/data https://edu.postgrespro.ru/demo-20250901-2y.sql.gz
```
